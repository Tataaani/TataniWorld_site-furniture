# 🪑 Интерьер —  интернет-магазин мебели

Современный и стильный многостраничный сайт для мебельного магазина. Проект выполнен с использованием чистой верстки, приятной цветовой гаммы и качественных интерьерных фотографий. 

## ✨ Главные особенности проекта
* **Многостраничная структура:** полноценный интерфейс интернет-магазина, включающий главную страницу, каталог и корзину.
* **Каталог товаров с фильтрацией:** продуманный интерфейс каталога со всплывающим окном тонкой настройки (фильтры по цене, цвету, срокам доставки и типу комнаты).
* **Функциональная корзина:** сверстан список добавленных товаров, кнопки для изменения количества, удаления и полной очистки корзины.
* **Форма оформления заказа:** удобный боковой блок для ввода данных покупателя (имя, телефон, адрес) и подсчета итоговой стоимости.
* **Эстетичный минимализм:** дизайн ориентирован на демонстрацию товаров премиум-качества без лишнего визуального шума.

## 🛠 Технологии в проекте
* **HTML5** — семантическая структура всех страниц интернет-магазина.
* **CSS3** — стилизация элементов, построение сложных сеток (Flexbox / Grid), оформление карточек товаров и всплывающего окна фильтров.

## 📂 Структура проекта
Сайт состоит из трех основных страниц:
1. **Главная страница (`index.html`):** приветственный баннер, плитка категорий комнат, акции со скидками и подвал с подпиской.
2. **Каталог товаров (`catalog.html`):** сетка всех товаров, элементы сортировки и поп-ап окно детальной фильтрации.
3. **Корзина (`cart.html`):** список покупок, блок оформления заказа и рекомендации «Специальные предложения».

## 🚀 Как запустить проект
1. Скопируйте репозиторий или скачайте архив с проектом.
2. Откройте файл `index.html` в любом современном браузере для старта.
